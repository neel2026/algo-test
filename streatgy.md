Critical ATM Options Factors 
1. Theta Decay (Time Decay) 
Code 
⚠️ ATM options lose value every minute 
→ AVOID buying options after 2:00 PM 
→ Best entries: 9:20 AM – 1:00 PM 
→ Never hold ATM overnight 
   (theta destroys premium) 
2. Expiry Awareness 
Code 
Weekly Expiry = Thursday (Nifty) 

Monday/Tuesday → Buy ATM freely 
                  (4-5 days left, less decay) 

Wednesday      → Be selective 
                  (decay accelerates) 

Thursday       → AVOID buying ATM 
                  (extreme theta risk) 
                  Only scalp with tight SL 
3. IV (Implied Volatility) 
Code 
High IV (VIX > 15) → 
→ Premiums expensive 
→ Smaller quantity 
→ Wider targets 

Low IV (VIX < 13) → 
→ Premiums cheap 
→ Better risk/reward 
→ Normal quantity 
4. Premium-Based Stop Loss 
Code 
Instead of point-based SL: 

Conservative → Exit if premium 
               drops 30% from entry 

Aggressive   → Exit if premium 
               drops 50% from entry 

Example: 
Buy 23,650 CE at ₹120 
→ Conservative SL = ₹84 
→ Aggressive SL   = ₹60
